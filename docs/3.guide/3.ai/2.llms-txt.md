## Что такое LLMs.txt?

LLMs.txt — структурированный формат документации для больших языковых моделей. В Nuxt есть файлы LLMs.txt с обзором фреймворка, чтобы ИИ-инструментам было проще помогать с разработкой.

Они заточены под потребление моделями: концепции, API, паттерны и практики.

## Доступные маршруты

- **`/llms.txt`** — структурированный обзор страниц документации и ссылок (~5K токенов)
- **`/llms-full.txt`** — полная документация: гайды, API, блог, деплой (~1M+ токенов)

## Какой файл выбрать

::note{icon="i-lucide-info"}
**Большинству достаточно `/llms.txt`** — там essentials под стандартное окно контекста.

`/llms-full.txt` — только если нужны детали реализации и у инструмента большой контекст (200K+ токенов).
::

## Важно при использовании

::warning{icon="i-lucide-alert-triangle"}
**Символ `@` нужно вводить вручную** — в Cursor/Windsurf при копировании контекст-ссылка может не сработать.
::

## ИИ-инструменты

### Cursor

Указывайте URL LLMs.txt в вопросах или добавляйте в контекст через `@docs`.

[Cursor Web и Docs Search](https://cursor.com/docs/context/mentions)

### Windsurf

`@docs` для URL LLMs.txt или правила в workspace с этими ссылками.

[Windsurf Web и Docs Search](https://docs.windsurf.com/windsurf/cascade/web-search)

### Другие инструменты

Любой клиент с поддержкой LLMs.txt может использовать эти маршруты.

Примеры запросов:

- "Используй документацию Nuxt с https://nuxt.com/llms.txt"
- "Следуй полным гайдлайнам Nuxt с https://nuxt.com/llms-full.txt"
