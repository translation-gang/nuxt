## Что такое LLMs.txt?

LLMs.txt — формат структурированной документации для больших языковых моделей. Nuxt публикует файлы LLMs.txt с обзором фреймворка, чтобы ИИ-инструментам было проще понимать контекст и помогать в разработке.

Файлы рассчитаны на потребление моделями и содержат сведения о концепциях, API, шаблонах использования и лучших практиках.

## Доступные маршруты

- **`/llms.txt`** — структурированный обзор страниц документации и ссылок (~5K токенов)
- **`/llms-full.txt`** — полный текст: раздел «с чего начать», API, блог, развёртывание (~1M+ токенов)

## Какой файл выбрать

::note{icon="i-lucide-info"}
**Большинству достаточно `/llms.txt`** — там сжатое содержание под стандартное окно контекста.

`/llms-full.txt` — только если нужны детали реализации и у инструмента большой контекст (200K+ токенов).
::

## Важно

::warning{icon="i-lucide-alert-triangle"}
**Символ `@` нужно вводить вручную** в Cursor, Windsurf и т.п. При вставке из буфера контекстные упоминания могут не сработать.
::

## ИИ-инструменты

### Cursor

Укажите URL LLMs.txt в запросах или добавьте их в контекст через `@docs`.

[Cursor: веб-поиск и поиск по документации](https://cursor.com/docs/context/mentions)

### Windsurf

`@docs` и постоянные правила в workspace с URL LLMs.txt.

[Windsurf: веб-поиск и поиск по документации](https://docs.windsurf.com/windsurf/cascade/web-search)

### Другие

Любой клиент с поддержкой LLMs.txt может использовать эти маршруты.

Примеры формулировок:

- «Используй документацию Nuxt с https://nuxt.com/llms.txt»
- «Следуй полным гайдлайнам Nuxt: https://nuxt.com/llms-full.txt»
