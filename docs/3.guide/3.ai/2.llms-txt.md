## Что такое LLMs.txt?

LLMs.txt — формат структурированной документации для больших языковых моделей (LLM). Nuxt публикует файлы LLMs.txt с полным обзором фреймворка, чтобы ИИ-инструментам было проще понимать контекст и помогать в разработке на Nuxt.

Файлы рассчитаны на потребление моделями и содержат структурированные сведения о концепциях, API, шаблонах использования и лучших практиках.

## Доступные маршруты

Маршруты LLMs.txt для подключения ИИ к документации:

- **`/llms.txt`** — структурированный обзор страниц документации и ссылок (~5K токенов)
- **`/llms-full.txt`** — развёрнутая документация: «с чего начать», справочник API, блог, развёртывание (~1M+ токенов)

## Какой файл выбрать

::note{icon="i-lucide-info"}
**В большинстве случаев начните с `/llms.txt`** — там всё необходимое и размер подходит под обычные окна контекста.

Используйте `/llms-full.txt`, только если нужны подробности реализации и ваш инструмент поддерживает большой контекст (200K+ токенов).
::

## Важные замечания по использованию

::warning{icon="i-lucide-alert-triangle"}
**Символ `@` нужно вводить вручную** — в Cursor, Windsurf и аналогичных клиентах при копировании из буфера упоминание контекста может не распознаться.
::

## Использование с ИИ-инструментами

### Cursor

Nuxt предоставляет специализированные файлы LLMs.txt, которые можно подключать в Cursor для более точной помощи по Nuxt.

#### Как использовать

1. **Прямая ссылка**: укажите URL LLMs.txt в запросе
2. Добавьте эти URL в контекст проекта через `@docs`

[Подробнее о веб-поиске и поиске по документации в Cursor](https://cursor.com/docs/context/mentions)

### Windsurf

Windsurf может напрямую обращаться к файлам LLMs.txt Nuxt, чтобы лучше понимать использование фреймворка.

#### LLMs.txt в Windsurf

- Используйте `@docs` для ссылки на конкретные URL LLMs.txt
- Создайте постоянные правила в workspace со ссылками на эти URL

[Подробнее о веб-поиске и документации в Windsurf](https://docs.windsurf.com/windsurf/cascade/web-search)

### Другие ИИ-инструменты

Любой клиент с поддержкой LLMs.txt может использовать эти маршруты.

#### Примеры для ChatGPT, Claude и других LLM

- «Используй документацию Nuxt с https://nuxt.com/llms.txt»
- «Следуй полным рекомендациям Nuxt: https://nuxt.com/llms-full.txt»
