## Что такое LLMs.txt?

LLMs.txt — формат структурированной документации для больших языковых моделей (LLM). Nuxt предоставляет файлы LLMs.txt с информацией о фреймворке, чтобы ИИ-инструменты могли помогать в разработке на Nuxt.

Файлы оптимизированы для потребления ИИ и содержат структурированные данные о концепциях, API, паттернах использования и лучших практиках.

## Доступные маршруты

Маршруты LLMs.txt для доступа к документации:

- **`/llms.txt`** — структурированный обзор страниц документации и ссылок (~5K токенов)
- **`/llms-full.txt`** — полная документация: гайды по началу работы, API, блог, развёртывание (~1M+ токенов)

## Какой файл выбрать

::note{icon="i-lucide-info"}
**Большинству достаточно `/llms.txt`** — в нём есть основное и он помещается в обычное окно контекста LLM.

Используйте `/llms-full.txt` только если нужны полные детали реализации и ваш ИИ-инструмент поддерживает большие контексты (200K+ токенов).
::

## Важно при использовании

::warning{icon="i-lucide-alert-triangle"}
**Символ @ нужно вводить вручную** — в Cursor, Windsurf и т.п. вводите `@` с клавиатуры. При вставке из буфера инструмент может не распознать его как ссылку на контекст.
::

## Использование в ИИ-инструментах

### Cursor

В Cursor можно подключать LLMs.txt Nuxt для лучших подсказок по разработке.

#### Как использовать

1. **Прямая ссылка**: указывайте URL LLMs.txt в вопросе
2. Добавляйте эти URL в контекст проекта через `@docs`

[Подробнее: Cursor Web and Docs Search](https://cursor.com/docs/context/symbols)

### Windsurf

Windsurf может обращаться к LLMs.txt Nuxt для понимания фреймворка и практик.

#### LLMs.txt в Windsurf

- Используйте `@docs` и укажите нужные URL LLMs.txt
- Создавайте постоянные правила с этими URL в рабочей области

[Подробнее: Windsurf Web and Docs Search](https://docs.windsurf.com/windsurf/cascade/web-search)

### Другие ИИ-инструменты

Любой инструмент с поддержкой LLMs.txt может использовать эти маршруты для работы с Nuxt.

#### Примеры для ChatGPT, Claude и др.

- «Используй документацию Nuxt с https://nuxt.com/llms.txt»
- «Следуй полным гайдам Nuxt с https://nuxt.com/llms-full.txt»
