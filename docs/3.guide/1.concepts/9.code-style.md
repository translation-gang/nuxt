## ESLint

Рекомендуемый путь — модуль [`@nuxt/eslint`](https://eslint.nuxt.com/packages/module): он настроит ESLint с учётом структуры проекта.

:::callout{icon="i-lucide-lightbulb"}
Модуль рассчитан на [новый flat config ESLint](https://eslint.org/docs/latest/use/configure/configuration-files) — [формат по умолчанию с ESLint v9](https://eslint.org/blog/2024/04/eslint-v9.0.0-released/). Для legacy `.eslintrc` нужна [ручная настройка с `@nuxt/eslint-config`](https://eslint.nuxt.com/packages/config#customizing-the-config). Лучше перейти на flat config.
:::

## Быстрая установка

```bash
npx nuxt module add eslint
```

Запустите приложение — в корне появится `eslint.config.mjs`; при необходимости отредактируйте.

Подробнее — [документация Nuxt ESLint](https://eslint.nuxt.com/packages/module).
