## ESLint

Рекомендуемый способ — включить ESLint через модуль [`@nuxt/eslint`](https://eslint.nuxt.com/packages/module). Он настроит ESLint с учётом структуры проекта.

:::callout{icon="i-lucide-lightbulb"}
Модуль рассчитан на [новый плоский формат конфигурации ESLint](https://eslint.org/docs/latest/use/configure/configuration-files), который [стандартен с ESLint v9](https://eslint.org/blog/2024/04/eslint-v9.0.0-released/). При использовании старого формата `.eslintrc` потребуется [ручная настройка с `@nuxt/eslint-config`](https://eslint.nuxt.com/packages/config#customizing-the-config). Рекомендуется перейти на плоский формат.
:::

## Быстрая настройка

```bash
npx nuxt module add eslint
```

После запуска приложения Nuxt в корне проекта будет создан файл `eslint.config.mjs`. Его можно настроить под себя.

Подробнее о модуле и настройке — в [документации Nuxt ESLint](https://eslint.nuxt.com/packages/module).
