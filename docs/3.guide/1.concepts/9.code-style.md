## ESLint

Рекомендуемый подход для Nuxt — включить поддержку ESLint с помощью модуля [`@nuxt/eslint`](https://eslint.nuxt.com/packages/module), который настроит для вас конфигурацию ESLint с учётом особенностей проекта.

:::callout{icon="i-lucide-lightbulb"}
Модуль рассчитан на [новый формат плоской конфигурации ESLint (flat config)](https://eslint.org/docs/latest/use/configure/configuration-files), который с [ESLint v9](https://eslint.org/blog/2024/04/eslint-v9.0.0-released/) используется по умолчанию. Если вы используете устаревший формат `.eslintrc`, потребуется [настроить всё вручную через `@nuxt/eslint-config`](https://eslint.nuxt.com/packages/config#customizing-the-config). Рекомендуем перейти на плоскую конфигурацию, чтобы не отставать от развития экосистемы.
:::

## Быстрая настройка

```bash
npx nuxt module add eslint
```

Запустите своё приложение Nuxt, и файл `eslint.config.mjs` будет сгенерирован в корневой директории вашего проекта. Настройте его под свои задачи.

Вы можете узнать больше о модуле и настройках в [документации Nuxt ESLint](https://eslint.nuxt.com/packages/module).
