::tip
Система хуков реализована на [unjs/hookable](https://github.com/unjs/hookable).
::

## Хуки Nuxt (время сборки)

Доступны в [модулях Nuxt](/docs/4.x/guide/modules) и контексте сборки.

### В nuxt.config.ts

```ts [nuxt.config.ts]
export default defineNuxtConfig({
  hooks: {
    close: () => { },
  },
})
```

### В модулях Nuxt

```js
import { defineNuxtModule } from '@nuxt/kit'

export default defineNuxtModule({
  setup (options, nuxt) {
    nuxt.hook('close', async () => { })
  },
})
```

## Хуки приложения (рантайм)

Используются в основном в [плагинах Nuxt](/docs/4.x/directory-structure/app/plugins) для подключения к жизненному циклу рендеринга, а также во Vue-композаблах.

```ts [app/plugins/test.ts]
export default defineNuxtPlugin((nuxtApp) => {
  nuxtApp.hook('page:start', () => {
    /* ваш код */
  })
})
```
