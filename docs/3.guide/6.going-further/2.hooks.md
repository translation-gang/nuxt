## Хуки Nuxt (время сборки)

Доступны в [модулях Nuxt](/docs/4.x/guide/modules) и контексте сборки.

### В nuxt.config.ts

```ts [nuxt.config.ts]
export default defineNuxtConfig({
  hooks: {
    close: () => { },
  },
})
```

### В модулях Nuxt

```js
import { defineNuxtModule } from '@nuxt/kit'

export default defineNuxtModule({
  setup (options, nuxt) {
    nuxt.hook('close', async () => { })
  },
})
```

## Хуки приложения (рантайм)

Используются в основном в [плагинах Nuxt](/docs/4.x/directory-structure/app/plugins) для подключения к жизненному циклу рендеринга, а также во Vue-композаблах.

```ts [app/plugins/test.ts]
export default defineNuxtPlugin((nuxtApp) => {
  nuxtApp.hook('page:start', () => {
    /* ваш код */
  })
})
```

## Серверные хуки (рантайм)

Доступны в [серверных плагинах](/docs/4.x/directory-structure/server#server-plugins) для подключения к поведению Nitro в рантайме.

```ts [~~/server/plugins/test.ts]
import { definePlugin } from 'nitro'

export default definePlugin((nitroApp) => {
  nitroApp.hooks.hook('render:html', (html, { event }) => {
    console.log('render:html', html)
    html.bodyAppend.push('<hr>Appended by custom plugin')
  })

  nitroApp.hooks.hook('render:response', (response, { event }) => {
    console.log('render:response', response)
  })
})
```

## Добавление своих хуков

Поддержку собственных хуков можно добавить, расширив интерфейсы хуков Nuxt.

```ts
import type { HookResult } from '@nuxt/schema'

declare module '#app' {
  interface RuntimeNuxtHooks {
    'your-nuxt-runtime-hook': () => HookResult
  }
  interface NuxtHooks {
    'your-nuxt-hook': () => HookResult
  }
}

declare module 'nitro/types' {
  interface NitroRuntimeHooks {
    'your-nitro-hook': () => void
  }
}
```
