Часть возможностей Nuxt включается по желанию или отключается под ваши задачи.

## `features`

### devLogs

Передача серверных логов на клиент в режиме разработки. Обработка в хуке `dev:ssr-logs`. По умолчанию включено в dev (если не активен test mode). При значении `silent` логи не выводятся в консоль браузера.

### inlineStyles

Инлайнить стили в HTML при рендеринге. Доступно при использовании Vite. Можно передать функцию `(id) => boolean` по пути компонента. По умолчанию `(id) => id.includes('.vue')`.

Остальные опции секции `features` см. в [документации API](/docs/4.x/api/nuxt-config#features).
