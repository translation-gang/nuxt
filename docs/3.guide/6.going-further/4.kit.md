Nuxt Kit — набор композабельных утилит для работы с [хуками Nuxt](/docs/4.x/api/advanced/hooks), [интерфейсом Nuxt](/docs/4.x/guide/going-further/internals#the-nuxt-interface) и разработки [модулей Nuxt](/docs/4.x/guide/modules).

::read-more{to="/docs/4.x/api/kit"}
Все утилиты Nuxt Kit в справочнике API.
::

## Использование

### Зависимость

Установите актуальный Nuxt Kit в секции `dependencies` в `package.json`. Рекомендуется явно указывать `@nuxt/kit`, даже если он уже тянется через Nuxt.

::note
`@nuxt/kit` и `@nuxt/schema` — ключевые зависимости Nuxt. При отдельной установке версии `@nuxt/kit` и `@nuxt/schema` должны быть не ниже версии `nuxt`, иначе возможно непредсказуемое поведение.
::

```json [package.json]
{
  "dependencies": {
    "@nuxt/kit": "npm:@nuxt/kit-nightly@latest"
  }
}
```

### Импорт утилит

```ts [test.mjs]
import { useNuxt } from '@nuxt/kit'
```

:read-more{title="Справочник Kit" to="/docs/4.x/api/kit"}

::note
Утилиты Nuxt Kit доступны только в модулях и не предназначены для runtime (компоненты, композаблы Vue, страницы, плагины или серверные маршруты).
::

Nuxt Kit — [только ESM](/docs/4.x/guide/concepts/esm), вызов `require('@nuxt/kit')` **невозможен**. В контексте CommonJS используйте динамический импорт:

```ts [test.cjs]
// This does NOT work!
// const kit = require('@nuxt/kit')
async function main () {
  const kit = await import('@nuxt/kit')
}
main()
```
