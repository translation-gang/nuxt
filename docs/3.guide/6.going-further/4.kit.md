Nuxt Kit — композабл-утилиты для работы с [хуками Nuxt](/docs/4.x/api/advanced/hooks), [интерфейсом Nuxt](/docs/4.x/guide/going-further/internals#the-nuxt-interface) и разработки [модулей Nuxt](/docs/4.x/guide/modules).

::read-more{to="/docs/4.x/api/kit"}
Справка по утилитам Nuxt Kit.
::

## Использование

### Установка

Добавьте `@nuxt/kit` в `dependencies` в `package.json`. Желательно явно указывать пакет даже если он уже тянется Nuxt.

::note
`@nuxt/kit` и `@nuxt/schema` — ключевые зависимости Nuxt. При отдельной установке версии должны быть не ниже версии `nuxt`.
::

### Импорт утилит

```ts [test.mjs]
import { useNuxt } from '@nuxt/kit'
```

Утилиты Kit доступны только в модулях; их **не** импортируют в рантайм-коде (компоненты, композаблы, страницы, плагины, серверные маршруты).

Nuxt Kit — [только ESM](/docs/4.x/guide/concepts/esm), то есть `require('@nuxt/kit')` не поддерживается. В CommonJS-контексте используйте динамический импорт.
