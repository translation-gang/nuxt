Nuxt Kit даёт композабл-утилиты для работы с [хуками Nuxt](/docs/4.x/api/advanced/hooks), [интерфейсом Nuxt](/docs/4.x/guide/going-further/internals#the-nuxt-interface) и разработки [модулей Nuxt](/docs/4.x/guide/modules).

::read-more{to="/docs/4.x/api/kit"}
Все утилиты Nuxt Kit.
::

## Использование

### Установка зависимости

Установите актуальный Nuxt Kit, добавив пакет в секцию `dependencies` в `package.json`. Лучше всегда явно указывать `@nuxt/kit`, даже если он уже тянется Nuxt.

::note
`@nuxt/kit` и `@nuxt/schema` — ключевые зависимости Nuxt. При отдельной установке следите, чтобы версии `@nuxt/kit` и `@nuxt/schema` были не ниже версии `nuxt`, иначе возможны сюрпризы.
::

```json [package.json]
{
  "dependencies": {
    "@nuxt/kit": "npm:@nuxt/kit-nightly@latest"
  }
}
```

### Импорт утилит Kit

```ts [test.mjs]
import { useNuxt } from '@nuxt/kit'
```

:read-more{to="/docs/4.x/api/kit"}

::note
Утилиты Nuxt Kit предназначены для модулей и не должны импортироваться в runtime (компоненты, Vue-композаблы, страницы, плагины или серверные маршруты).
::

Nuxt Kit — [пакет только в формате ESM](/docs/4.x/guide/concepts/esm), то есть **`require('@nuxt/kit')` недоступен**. В CommonJS используйте динамический импорт:

```ts [test.cjs]
// Так НЕ работает!
// const kit = require('@nuxt/kit')
async function main () {
  const kit = await import('@nuxt/kit')
}
main()
```
