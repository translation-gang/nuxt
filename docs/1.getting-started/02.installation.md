## Попробовать онлайн

Если вы хотите попробовать Nuxt в браузере без локальной установки, воспользуйтесь нашими онлайн-песочницами:

::card-group
  :card{title="Открыть в StackBlitz" icon="i-simple-icons-stackblitz" to="https://nuxt.new/s/v4" target="_blank"}
  :card{title="Открыть в CodeSandbox" icon="i-simple-icons-codesandbox" to="https://nuxt.new/c/v4" target="_blank"}
::

Или следуйте шагам ниже, чтобы запустить проект на своём компьютере.

#### Требования

- **Node.js** - [`20.x`](https://nodejs.org/en) или более новые (но мы рекомендуем [активную LTS-версию](https://github.com/nodejs/release#release-schedule))
- **Редактор кода** - Нет требований к IDE, но мы рекомендуем [Visual Studio Code](https://code.visualstudio.com/) с [официальным расширением Vue](https://marketplace.visualstudio.com/items?itemName=Vue.volar) (ранее известным как Volar) или [WebStorm](https://www.jetbrains.com/webstorm/), который, наряду с [другими JetBrains IDEs](https://www.jetbrains.com/ides/), предлагает отличную поддержку Nuxt прямо из коробки.
- **Терминал** — для запуска команд Nuxt CLI

::note
  ::details
  :summary[Дополнительные указания для оптимального запуска:]
  - **Node.js**: используйте чётную версию (18, 20 и т. д.)
  - **Nuxtr**: расширение сообщества — [Nuxtr для VS Code](https://marketplace.visualstudio.com/items?itemName=Nuxtr.nuxtr-vscode)
  - **WSL**: на Windows при медленном HMR имеет смысл попробовать [WSL (подсистема Linux)](https://docs.microsoft.com/en-us/windows/wsl/install) — часто это улучшает производительность.
  ::
::

Откройте терминал (в [Visual Studio Code](https://code.visualstudio.com) удобно пользоваться [встроенным терминалом](https://code.visualstudio.com/docs/editor/integrated-terminal)) и выполните команды для создания проекта:

::code-group{sync="pm"}

```bash [npm]
npm create nuxt@latest <project-name> -- -t v3
```

```bash [yarn]
yarn create nuxt@latest <project-name> -t v3
```

```bash [pnpm]
pnpm create nuxt@latest <project-name> -t v3
```

```bash [bun]
bun create nuxt@latest <project-name> -t v3
```

```bash [deno]
deno -A npm:create-nuxt@latest <project-name> -t v3
```

::

::tip
Готовые стартовые шаблоны и темы есть на [nuxt.new](https://nuxt.new) — следуйте инструкциям на сайте.
::

Откройте папку проекта в Visual Studio Code:

```bash [Terminal]
code <имя-проекта>
```

Или измените текущую директорию на директорию вашего проекта:

```bash
cd <имя-проекта>
```

## Сервер разработки

::code-group{sync="pm"}

```bash [npm]
npm run dev -- -o
```

```bash [yarn]
yarn dev --open
```

```bash [pnpm]
pnpm dev -o
```

```bash [bun]
bun run dev -o

# Чтобы в dev использовать рантайм Bun:
# bun --bun run dev -o
```

```bash [deno]
deno run dev -o
```
::

::tip{icon="i-lucide-circle-check"}
Отлично! Теперь в окне браузера должно автоматически открыться <http://localhost:3000>.
::
