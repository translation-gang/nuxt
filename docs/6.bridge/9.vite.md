---
title: Vite
description: Подключение Vite к приложению Nuxt 2 через Nuxt Bridge (после настройки Nitro).
---

::warning
Чтобы включить Vite (опция `vite: true`), сначала настройте [Nitro](/docs/bridge/nitro).
::

## Удалите модули

- Удалите модуль `nuxt-vite`: ту же функциональность даёт Nuxt Bridge

## Обновите конфигурацию

```ts [nuxt.config.ts]
import { defineNuxtConfig } from '@nuxt/bridge'

export default defineNuxtConfig({
  bridge: {
    vite: true,
    nitro: true,
  },
})
```

## Настройка Vite

```ts [nuxt.config.ts]
import { defineNuxtConfig } from '@nuxt/bridge'

export default defineNuxtConfig({
  vite: {
    // опции Vite
  },
})
```

Сводка флагов Bridge — на странице [конфигурации Nuxt Bridge](/docs/bridge/configuration).
