```bash [Terminal]
npx nuxi upgrade [--force|-f]
```

Команда `upgrade` обновляет Nuxt 3 до последней версии.

Параметр      | По умолчанию | Описание
--------------|--------------|------------------------------------------------------
`--force, -f` | `false`      | Удаляет `node_modules` и `.lock-файл` перед обновлением.
