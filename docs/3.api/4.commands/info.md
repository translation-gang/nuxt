```bash [Terminal]
npx nuxi info [rootDir]
```

Команда `info` выводит сведения о текущем или указанном проекте Nuxt.

Параметр  | По умолчанию | Описание
----------|--------------|-----------------------------------------
`rootDir` | `.`          | Директория целевого приложения.
