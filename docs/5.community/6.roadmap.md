## Цикл релизов

С января 2023 года у Nuxt стабильный цикл релизов по [semver](https://semver.org): мажорные релизы фреймворка примерно раз в год, патчи — примерно еженедельно, минорные — примерно ежемесячно. Breaking changes только в опциях, явно помеченных как `experimental`.

Для Nuxt 4 и Nuxt 5 планируется небольшое изменение: Nuxt 4 — стабильный релиз со всеми фичами `compatibilityVersion: 4`, за ним вскоре последует Nuxt 5 с переходом на Nitro v3 и другими изменениями.

Так breaking changes разнесены по этапам, что упрощает тестирование экосистемы и миграции.

### Поддержка версий Nuxt

Мы обязуемся поддерживать каждую мажорную версию Nuxt минимум шесть месяцев после выхода следующей мажорной и предоставлять путь миграции для текущих пользователей.

### Текущие пакеты

Текущая активная версия [Nuxt](https://nuxt.com) — **v4**, доступна на npm как `nuxt` с тегом `latest`.

Nuxt 3 продолжит получать обновления обслуживания (багфиксы и патчи безопасности) до конца июля 2026 года.

У каждой активной версии есть ночные сборки. Подробнее — [канал ночных релизов](/docs/4.x/guide/going-further/nightly-release-channel).

### Статусы поддержки

| Статус      | Описание                                                                   |
|-------------|----------------------------------------------------------------------------|
| Unsupported | Версия больше не поддерживается и не получает патчи безопасности           |
| Maintenance | Версия получает только патчи безопасности                                 |
| Stable      | Активная разработка и патчи безопасности                                 |
| Development | Версия может быть нестабильной                                            |
| Scheduled   | Версия ещё не вышла, но запланирована                                     |
