## Цикл релизов

С января 2023 года у Nuxt единый цикл релизов по [semver](https://semver.org). Ориентир: мажор раз в год, патчи — примерно раз в неделю, миноры — примерно раз в месяц. Без breaking changes, кроме опций, явно помеченных как `experimental`.

Для Nuxt 4 и 5 план небольшой корректировки. Nuxt 4 — релиз про стабильность со всеми фичами `compatibilityVersion: 4`, затем вскоре Nuxt 5 с обновлением до Nitro v3 и дополнительными изменениями.

Так breaking changes разбиваются на этапы: проще тестировать экосистему и мигрировать.

### Поддержка версий Nuxt

Мы обязуемся поддерживать каждый мажор минимум шесть месяцев после выхода следующего мажора и давать путь обновления для текущих пользователей.

### Текущие пакеты

Активная версия [Nuxt](https://nuxt.com) — **v4**, в npm как пакет `nuxt` с тегом `latest`.

Nuxt 3 будет получать обслуживающие обновления (исправления и патчи безопасности) до конца июля 2026 года.

У каждой активной версии есть ночные сборки, генерируемые автоматически. Подробнее — [документация по ночному каналу](/docs/4.x/guide/going-further/nightly-release-channel).

### Статусы поддержки

| Статус            | Описание                                                                      |
|-------------------|-------------------------------------------------------------------------------|
| Не поддерживается | Версия не поддерживается, патчи безопасности не выходят                       |
| Обслуживание      | Только патчи безопасности                                                     |
| Стабильная        | Активная разработка и патчи безопасности                                      |
| Разработка        | Версия может быть нестабильной                                                |
| Запланирована     | Версии ещё нет, но она запланирована                                          |
