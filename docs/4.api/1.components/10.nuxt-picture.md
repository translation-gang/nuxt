`<NuxtPicture>` — прямая замена нативного тега `<picture>`.

Использование `<NuxtPicture>` почти совпадает с [`<NuxtImg>`](/docs/3.x/api/components/nuxt-img), но при возможности подключает современные форматы, например `webp`.

О теге `<picture>` см. [документацию MDN](https://developer.mozilla.org/ru/docs/Web/HTML/Element/picture).

Перед использованием установите и подключите модуль Nuxt Image:

```bash [Terminal]
npx nuxt module add image
```

## Использование

Передавайте те же параметры, что и для `<NuxtImg>`: компонент сформирует разметку `<picture>` и при поддержке провайдера добавит варианты в современных форматах.
