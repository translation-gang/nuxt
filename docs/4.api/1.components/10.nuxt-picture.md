`<NuxtPicture>` — замена нативному тегу `<picture>`.

Использование почти как у [`<NuxtImg>`](/docs/4.x/api/components/nuxt-img), плюс при возможности отдаются современные форматы вроде `webp`.

О теге [`<picture>` на MDN](https://developer.mozilla.org/en-US/docs/Web/HTML/Reference/Elements/picture).

## Установка

Установите и подключите модуль Nuxt Image:

```bash [Terminal]
npx nuxt module add image
```
