`<NuxtPicture>` — замена нативного тега `<picture>`.

Использование почти такое же, как у [`<NuxtImg>`](/docs/4.x/api/components/nuxt-img); дополнительно позволяет отдавать современные форматы (например `webp`) при поддержке браузером.

Подробнее о [теге `<picture>` на MDN](https://developer.mozilla.org/en-US/docs/Web/HTML/Reference/Elements/picture).

## Установка

Нужно установить и подключить модуль Nuxt Image:

```bash [Terminal]
npx nuxt module add image
```
