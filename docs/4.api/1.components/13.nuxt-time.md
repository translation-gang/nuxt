::important
Доступен в Nuxt v3.17+.
::

`<NuxtTime>` выводит дату и время с учётом локали и корректной семантикой `<time>`. Рендер на сервере и клиенте согласован, без ошибок гидратации.

## Использование

Используйте `<NuxtTime>` в любом месте приложения:

```vue
<template>
  <NuxtTime :datetime="Date.now()" />
</template>
```

## Пропы

### `datetime`

- Тип: `Date | number | string`
- Обязательный: `true`

Значение даты и времени:
- объект `Date`
- число (timestamp)
- строка в формате ISO

```vue
<template>
  <NuxtTime :datetime="Date.now()" />
  <NuxtTime :datetime="new Date()" />
  <NuxtTime datetime="2023-06-15T09:30:00.000Z" />
</template>
```

### `locale`

- Тип: `string`
- Обязательный: `false`
- По умолчанию: локаль браузера или сервера по умолчанию

[BCP 47](https://datatracker.ietf.org/doc/html/rfc5646) для форматирования (например `'en-US'`, `'fr-FR'`, `'ja-JP'`):

```vue
<template>
  <NuxtTime
    :datetime="Date.now()"
    locale="fr-FR"
  />
</template>
```

### Пропы форматирования

Принимаются опции [Intl.DateTimeFormat](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Intl/DateTimeFormat/DateTimeFormat):

```vue
<template>
  <NuxtTime
    :datetime="Date.now()"
    year="numeric"
    month="long"
    day="numeric"
    hour="2-digit"
    minute="2-digit"
  />
</template>
```

Пример вывода: "April 22, 2025, 08:30 AM"

### `relative`

- Тип: `boolean`
- Обязательный: `false`
- По умолчанию: `false`

Относительное время через Intl.RelativeTimeFormat:

```vue
<template>
  <!-- Shows something like "5 minutes ago" -->
  <NuxtTime
    :datetime="Date.now() - 5 * 60 * 1000"
    relative
  />
</template>
```

### Пропы относительного форматирования

При `relative: true` доступны опции [Intl.RelativeTimeFormat](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Intl/RelativeTimeFormat/RelativeTimeFormat):

::warning
`style` зарезервирован; для стиля относительного времени используйте `relativeStyle`.
::

```vue
<template>
  <NuxtTime
    :datetime="Date.now() - 3 * 24 * 60 * 60 * 1000"
    relative
    numeric="auto"
    relative-style="long"
  />
</template>
```

Пример: "3 days ago" или "last Friday" в зависимости от `numeric`.

## Примеры

### Базовое использование

```vue
<template>
  <NuxtTime :datetime="Date.now()" />
</template>
```

### Свой формат

```vue
<template>
  <NuxtTime
    :datetime="Date.now()"
    weekday="long"
    year="numeric"
    month="short"
    day="numeric"
    hour="numeric"
    minute="numeric"
    second="numeric"
    time-zone-name="short"
  />
</template>
```

### Относительное время

```vue
<template>
  <div>
    <p>
      <NuxtTime
        :datetime="Date.now() - 30 * 1000"
        relative
      />
      <!-- 30 seconds ago -->
    </p>
    <p>
      <NuxtTime
        :datetime="Date.now() - 45 * 60 * 1000"
        relative
      />
      <!-- 45 minutes ago -->
    </p>
    <p>
      <NuxtTime
        :datetime="Date.now() + 2 * 24 * 60 * 60 * 1000"
        relative
      />
      <!-- in 2 days -->
    </p>
  </div>
</template>
```

### Своя локаль

```vue
<template>
  <div>
    <NuxtTime
      :datetime="Date.now()"
      locale="en-US"
      weekday="long"
    />
    <NuxtTime
      :datetime="Date.now()"
      locale="fr-FR"
      weekday="long"
    />
    <NuxtTime
      :datetime="Date.now()"
      locale="ja-JP"
      weekday="long"
    />
  </div>
</template>
```
