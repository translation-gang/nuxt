::important
Доступен в Nuxt v3.12+.
::

## Использование

Добавьте `<NuxtRouteAnnouncer/>` в [`app.vue`](/docs/4.x/directory-structure/app/app) или [`app/layouts/`](/docs/4.x/directory-structure/app/layouts), чтобы улучшить доступность: скринридеры узнают о смене заголовка страницы при навигации.

```vue [app/app.vue]
<template>
  <NuxtRouteAnnouncer />
  <NuxtLayout>
    <NuxtPage />
  </NuxtLayout>
</template>
```

## Слоты

В слот по умолчанию можно передать свой HTML или компоненты.

```vue
<template>
  <NuxtRouteAnnouncer>
    <template #default="{ message }">
      <p>{{ message }} was loaded.</p>
    </template>
  </NuxtRouteAnnouncer>
</template>
```

## Пропы

- `atomic`: если `true`, зачитывается весь контент при обновлении; если `false` — только изменения. (по умолчанию `false`)
- `politeness`: срочность объявления: `off` (отключить), `polite` (после паузы), `assertive` (прервать текущее). (по умолчанию `polite`)

::callout
Компонент необязательный. :br
Для полного контроля реализуйте свой на основе [исходного кода](https://github.com/nuxt/nuxt/blob/main/packages/nuxt/src/app/components/nuxt-route-announcer.ts).
::

::callout
К экземпляру announcer подключается композабл [`useRouteAnnouncer`](/docs/4.x/api/composables/use-route-announcer) — можно задать свой текст объявления.
::

::callout
Для изменений внутри страницы (валидация форм, тосты, загрузка и т.д.) используйте [`<NuxtAnnouncer>`](/docs/4.x/api/components/nuxt-announcer) и [`useAnnouncer`](/docs/4.x/api/composables/use-announcer).
::
