::important
Компонент доступен в Nuxt v3.12+.
::

## Использование

Добавьте `<NuxtRouteAnnouncer/>` в [`app.vue`](/docs/4.x/directory-structure/app/app) или в [`app/layouts/`](/docs/4.x/directory-structure/app/layouts), чтобы вспомогательные технологии (скринридеры) получали уведомления о смене заголовка страницы при навигации.

```vue [app/app.vue]
<template>
  <NuxtRouteAnnouncer />
  <NuxtLayout>
    <NuxtPage />
  </NuxtLayout>
</template>
```

## Слоты

Через слот по умолчанию можно передать свой HTML или компоненты.

```vue
<template>
  <NuxtRouteAnnouncer>
    <template #default="{ message }">
      <p>{{ message }} загружено.</p>
    </template>
  </NuxtRouteAnnouncer>
</template>
```

## Пропсы

- `atomic`: озвучивать только изменения или весь контент. `true` — полное озвучивание при обновлении, `false` — только изменения (по умолчанию `false`).
- `politeness`: срочность объявления — `off`, `polite` (по умолчанию) или `assertive`.

::callout
Компонент опционален. :br
Для полной кастомизации можно сделать свой на основе [исходного кода](https://github.com/nuxt/nuxt/blob/main/packages/nuxt/src/app/components/nuxt-route-announcer.ts).
::

::callout
Управление объявлениями — через композабл [useRouteAnnouncer](/docs/4.x/api/composables/use-route-announcer), в том числе для кастомного сообщения.
::
