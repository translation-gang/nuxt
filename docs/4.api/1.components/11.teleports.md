---
title: '<Teleport>'
description: Компонент <Teleport> телепортирует компонент в другое место в DOM.
---

::warning
Цель `to` в [`<Teleport>`](https://ru.vuejs.org/guide/built-ins/teleport.html) ожидает строку CSS-селектора или фактический узел DOM. В настоящее время Nuxt имеет поддержку SSR только для телепортов в `#teleports`, а для других целей на стороне клиента используется обёртка `<ClientOnly>`.
::

## Телепорт в контейнер `#teleports`

```vue
<template>
  <button @click="open = true">
    Открыть модальное окно
  </button>
  <Teleport to="#teleports">
    <div v-if="open" class="modal">
      <p>Привет из модального окна!</p>
      <button @click="open = false">
        Закрыть
      </button>
    </div>
  </Teleport>
</template>
```

## Телепорт на стороне клиента

```vue
<template>
  <ClientOnly>
    <Teleport to="#some-selector">
      <!-- контент -->
    </Teleport>
  </ClientOnly>
</template>
```
