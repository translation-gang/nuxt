---
title: '<Teleport>'
description: "Компонент <Teleport> переносит содержимое в другое место DOM."
---

::warning
Целевой элемент `to` у [`<Teleport>`](https://vuejs.org/guide/built-ins/teleport) — CSS-селектор или DOM-узел. В Nuxt SSR поддерживается только для `#teleports`; для других целей используйте обёртку `<ClientOnly>`.
::

## Teleport в #teleports

```vue
<template>
  <button @click="open = true">
    Открыть модалку
  </button>
  <Teleport to="#teleports">
    <div
      v-if="open"
      class="modal"
    >
      <p>Содержимое модалки</p>
      <button @click="open = false">
        Закрыть
      </button>
    </div>
  </Teleport>
</template>
```

## Teleport на клиенте

```vue
<template>
  <ClientOnly>
    <Teleport to="#some-selector">
      <!-- контент -->
    </Teleport>
  </ClientOnly>
</template>
```
