---
title: '<Teleport>'
description: Компонент <Teleport> переносит содержимое в другое место DOM.
---

::warning
Цель `to` у [`<Teleport>`](https://vuejs.org/guide/built-ins/teleport) — CSS-селектор или DOM-узел. В Nuxt SSR поддерживаются телепорты только в `#teleports`; для других целей на клиенте оберните в `<ClientOnly>`.
::

## Телепорт в body

```vue
<template>
  <button @click="open = true">
    Open Modal
  </button>
  <Teleport to="#teleports">
    <div
      v-if="open"
      class="modal"
    >
      <p>Hello from the modal!</p>
      <button @click="open = false">
        Close
      </button>
    </div>
  </Teleport>
</template>
```

## Телепорт только на клиенте

```vue
<template>
  <ClientOnly>
    <Teleport to="#some-selector">
      <!-- content -->
    </Teleport>
  </ClientOnly>
</template>
```
