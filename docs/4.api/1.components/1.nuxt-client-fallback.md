Nuxt предоставляет компонент `<NuxtClientFallback>`, который рендерит своё содержимое на клиенте, если при SSR у одного из дочерних элементов возникает ошибка.

::note{to="/docs/4.x/guide/going-further/experimental-features#clientfallback"}
Компонент экспериментальный; чтобы использовать его, включите опцию `experimental.clientFallback` в `nuxt.config`.
::

```vue [app/pages/example.vue]
<template>
  <div>
    <Sidebar />
    <!-- this component will be rendered on client-side -->
    <NuxtClientFallback fallback-tag="span">
      <Comments />
      <BrokeInSSR />
    </NuxtClientFallback>
  </div>
</template>
```

## События

- `@ssr-error`: событие при ошибке дочернего элемента в SSR. Срабатывает только на сервере.

  ```vue
  <template>
    <NuxtClientFallback @ssr-error="logSomeError">
      <!-- ... -->
    </NuxtClientFallback>
  </template>
  ```

## Пропы

- `placeholderTag` | `fallbackTag`: тег запасного варианта, если слот не удалось отрендерить на сервере.
  - **тип**: `string`
  - **по умолчанию**: `div`
- `placeholder` | `fallback`: запасное содержимое, если слот не удалось отрендерить.
  - **тип**: `string`
- `keepFallback`: сохранять запасное содержимое, если на сервере рендер не удался.
  - **тип**: `boolean`
  - **по умолчанию**: `false`

::warning{icon="i-ph-warning-duotone"}
Пропы `placeholder` и `fallback` выводят содержимое как сырой HTML. Не передавайте в них недоверенный пользовательский ввод — возможны XSS. Для динамического контента с экранированием используйте слоты `#fallback` или `#placeholder`.
::

```vue
<template>
  <!-- render <span>Hello world</span> server-side if the default slot fails to render -->
  <NuxtClientFallback
    fallback-tag="span"
    fallback="Hello world"
  >
    <BrokeInSSR />
  </NuxtClientFallback>
</template>
```

## Слоты

- `#fallback`: содержимое для показа на сервере, если слот не удалось отрендерить.

```vue
<template>
  <NuxtClientFallback>
    <!-- ... -->
    <template #fallback>
      <!-- this will be rendered on server side if the default slot fails to render in ssr -->
      <p>Hello world</p>
    </template>
  </NuxtClientFallback>
</template>
```
