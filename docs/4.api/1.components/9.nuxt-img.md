`<NuxtImg>` — замена нативному тегу `<img>`.

- Встроенный провайдер оптимизирует локальные и удалённые изображения
- `src` преобразуется в URL провайдера
- Автоматический ресайз по `width` и `height`
- Адаптивные размеры при опции `sizes`
- Нативный lazy loading и остальные атрибуты `<img>`

## Установка

Установите и подключите модуль Nuxt Image:

```bash [Terminal]
npx nuxt module add image
```

## Использование

`<NuxtImg>` выводит обычный `img` без обёртки. Используйте как `<img>`:

```html
<NuxtImg src="/nuxt-icon.png" />
```

Результат:

```html
<img src="/nuxt-icon.png" />
```
