## `updateAppConfig`

Обновляет конфигурацию приложения, которая попадёт в `nuxt.options.appConfig`. Модули используют это для значений по умолчанию, которые пользователь может переопределить. Обновление мержится с существующей конфигурацией через [`defu`](https://github.com/unjs/defu).

### Тип

```ts
function updateAppConfig (appConfig: Record<string, unknown>): void
```

### Пример

```ts
import { defineNuxtModule, updateAppConfig } from '@nuxt/kit'

export default defineNuxtModule({
  setup () {
    updateAppConfig({
      myModule: {
        option: 'value',
      },
    })
  },
})
```
