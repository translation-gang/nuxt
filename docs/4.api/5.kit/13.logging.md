Nuxt предоставляет экземпляр логгера для сообщений с дополнительными возможностями. `useLogger` возвращает этот экземпляр.

## `useLogger`

Возвращает экземпляр логгера. Внутри используется [consola](https://github.com/unjs/consola).

### Использование

```ts twoslash
import { defineNuxtModule, useLogger } from '@nuxt/kit'

export default defineNuxtModule({
  setup (options, nuxt) {
    const logger = useLogger('my-module')

    logger.info('Hello from my module!')
  },
})
```

### Тип

```ts
function useLogger (tag?: string, options?: Partial<ConsolaOptions>): ConsolaInstance
```

### Параметры

**`tag`**: метка, которая дописывается ко всем сообщениям и отображается справа у временной метки.

**`options`**: параметры конфигурации Consola.

### Примеры

```ts twoslash
import { defineNuxtModule, useLogger } from '@nuxt/kit'

export default defineNuxtModule({
  setup (options, nuxt) {
    const logger = useLogger('my-module', { level: options.quiet ? 0 : 3 })

    logger.info('Hello from my module!')
  },
})
```
