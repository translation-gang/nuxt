## `useRuntimeConfig`

Во время сборки даёт доступ к итоговой [runtime-конфигурации](/docs/4.x/guide/going-further/runtime-config) Nuxt.

### Тип

```ts
function useRuntimeConfig (): Record<string, unknown>
```

## `updateRuntimeConfig`

Обновляет runtime-конфигурацию: значения сливаются с существующими. Если Nitro уже инициализирован, вызывается HMR-событие для перезагрузки конфига Nitro.

### Тип

```ts
function updateRuntimeConfig (config: Record<string, unknown>): void | Promise<void>
```
