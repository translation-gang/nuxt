## `useRuntimeConfig`

На этапе сборки доступна итоговая [runtime-конфигурация](/docs/3.x/guide/going-further/runtime-config) Nuxt.

### Тип

```ts
function useRuntimeConfig (): Record<string, unknown>
```

## `updateRuntimeConfig`

Конфигурацию runtime можно обновить: новые значения сливаются с текущими. Если Nitro уже инициализирован, срабатывает HMR и перезагружается runtime-конфигурация Nitro.

### Тип

```ts
function updateRuntimeConfig (config: Record<string, unknown>): void | Promise<void>
```
