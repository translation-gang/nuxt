## `useRuntimeConfig`

На этапе сборки можно получить доступ к разрешённой [runtime-конфигурации](/docs/4.x/guide/going-further/runtime-config) Nuxt.

### Тип

```ts
function useRuntimeConfig (): Record<string, unknown>
```

## `updateRuntimeConfig`

Также можно обновить runtime-конфигурацию. Она сольётся с текущей; если Nitro уже инициализирован, будет вызвано HMR для перезагрузки runtime-конфига Nitro.

### Тип

```ts
function updateRuntimeConfig (config: Record<string, unknown>): void | Promise<void>
```
