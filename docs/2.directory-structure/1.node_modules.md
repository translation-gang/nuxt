Менеджер пакетов ([`npm`](https://docs.npmjs.com/cli/commands/npm/), [`yarn`](https://yarnpkg.com), [`pnpm`](https://pnpm.io/cli/install), [`bun`](https://bun.com/package-manager) или [`deno`](https://docs.deno.com/runtime/getting_started/installation/)) создаёт эту директорию для хранения зависимостей проекта.

::important
Эту директорию стоит добавить в [`.gitignore`](/docs/4.x/directory-structure/gitignore), чтобы не коммитить зависимости в репозиторий.
::
