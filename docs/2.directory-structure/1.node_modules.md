Менеджер пакетов ([`npm`](https://docs.npmjs.com/cli/commands/npm), [`yarn`](https://yarnpkg.com), [`pnpm`](https://pnpm.io/cli/install) или [`bun`](https://bun.sh/package-manager)) создаёт этот каталог для зависимостей проекта.

::important
Добавьте этот каталог в [`.gitignore`](/docs/3.x/directory-structure/gitignore/), чтобы не коммитить `node_modules`.
::
