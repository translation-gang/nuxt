Здесь обычно лежат:

- Стили (CSS, SASS и т. п.)
- Шрифты
- Изображения, которые не будут отдаваться из директории [`public/`](/docs/3.x/directory-structure/public/).

Если файлы нужно отдавать с сервера без участия сборщика, смотрите каталог [`public/`](/docs/3.x/directory-structure/public/).
