Каталог [`app/utils/`](/docs/4.x/directory-structure/app/utils) отделяет по смыслу обычные утилиты от Vue composables с автоимпортом.

## Использование

**Способ 1:** именованный экспорт

```ts twoslash [utils/index.ts]
export const { format: formatNumber } = Intl.NumberFormat('en-GB', {
  notation: 'compact',
  maximumFractionDigits: 1,
})
```

**Способ 2:** экспорт по умолчанию

```ts twoslash [utils/random-entry.ts or utils/randomEntry.ts]
// It will be available as randomEntry() (camelCase of file name without extension)
export default function (arr: Array<any>) {
  return arr[Math.floor(Math.random() * arr.length)]
}
```

Утилиты с автоимпортом доступны в `.js`, `.ts` и `.vue`.

```vue [app/app.vue]
<template>
  <p>{{ formatNumber(1234) }}</p>
</template>
```

::tip
Правила автоимпорта и сканирования `app/utils/` такие же, как у [`app/composables/`](/docs/4.x/directory-structure/app/composables).
::

::important
Эти утилиты доступны только во Vue-части приложения. :br
В [`server/`](/docs/4.x/directory-structure/server#server-utilities) автоимпортируются только `server/utils`.
::
