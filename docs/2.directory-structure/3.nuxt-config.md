Расширение файла конфигурации может быть `.js`, `.ts` или `.mjs`.

```ts twoslash [nuxt.config.ts]
export default defineNuxtConfig({
  // Моя конфигурация Nuxt
})
```

::tip
Хелпер `defineNuxtConfig` доступен глобально без импорта.
::

При желании можно импортировать его явно из `nuxt/config`:

```ts twoslash [nuxt.config.ts]
import { defineNuxtConfig } from 'nuxt/config'

export default defineNuxtConfig({
  // Моя конфигурация Nuxt
})
```

::read-more{to="/docs/4.x/api/configuration/nuxt-config"}
Все доступные опции — в документации **конфигурации Nuxt**.
::

При изменении основного конфига, [`.env`](/docs/4.x/directory-structure/env), [`.nuxtignore`](/docs/4.x/directory-structure/nuxtignore) и [`.nuxtrc`](/docs/4.x/directory-structure/nuxtrc) Nuxt перезапускается, чтобы применить новые настройки.
