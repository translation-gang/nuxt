Расширение файла `nuxt.config` может быть `.js`, `.ts` или `.mjs`.

```ts twoslash [nuxt.config.ts]
export default defineNuxtConfig({
  // My Nuxt config
})
```

::tip
Хелпер `defineNuxtConfig` доступен глобально без импорта.
::

При желании импортируйте `defineNuxtConfig` из `nuxt/config`:

```ts twoslash [nuxt.config.ts]
import { defineNuxtConfig } from 'nuxt/config'

export default defineNuxtConfig({
  // My Nuxt config
})
```

::read-more{to="/docs/4.x/api/configuration/nuxt-config"}
Все опции — в документации **конфигурации Nuxt**.
::

Чтобы конфигурация всегда была актуальной, Nuxt делает полный перезапуск при изменении основного конфигурационного файла и точечных файлов [`.env`](/docs/4.x/directory-structure/env), [`.nuxtignore`](/docs/4.x/directory-structure/nuxtignore) и [`.nuxtrc`](/docs/4.x/directory-structure/nuxtrc).
