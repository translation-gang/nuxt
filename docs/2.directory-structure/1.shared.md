Каталог `shared/` позволяет использовать общий код и во Vue-приложении, и на сервере Nitro.

::note
Каталог `shared/` доступен в Nuxt v3.14+.
::

::important
Код в `shared/` не может импортировать код Vue или Nitro.
::

## Использование

**Способ 1:** именованный экспорт

```ts twoslash [shared/utils/capitalize.ts]
export const capitalize = (input: string) => {
  return input[0] ? input[0].toUpperCase() + input.slice(1) : ''
}
```

**Способ 2:** экспорт по умолчанию

```ts twoslash [shared/utils/capitalize.ts]
export default function (input: string) {
  return input[0] ? input[0].toUpperCase() + input.slice(1) : ''
}
```

Далее можно использовать [автоимпортированные](/docs/4.x/directory-structure/shared) утилиты в приложении Nuxt и в каталоге `server/`.

```vue [app/app.vue]
<script setup lang="ts">
const hello = capitalize('hello')
</script>

<template>
  <div>
    {{ hello }}
  </div>
</template>
```

```ts [server/api/hello.get.ts]
export default defineEventHandler((event) => {
  return {
    hello: capitalize('hello'),
  }
})
```

## Как сканируются файлы

Автоимпорт затрагивает только файлы в `shared/utils/` и `shared/types/`. Вложенные подкаталоги этих папок не попадают в автоимпорт, пока вы не добавите их в `imports.dirs` и `nitro.imports.dirs`.

::tip
Принцип автоимпорта и сканирования `shared/utils` и `shared/types` такой же, как у [`app/composables/`](/docs/4.x/directory-structure/app/composables) и [`app/utils/`](/docs/4.x/directory-structure/app/utils).
::

:read-more{to="/docs/4.x/directory-structure/app/composables#how-files-are-scanned"}

```bash [Directory Structure]
-| shared/
---| capitalize.ts        # Not auto-imported
---| formatters
-----| lower.ts           # Not auto-imported
---| utils/
-----| lower.ts           # Auto-imported
-----| formatters
-------| upper.ts         # Not auto-imported
---| types/
-----| bar.ts             # Auto-imported
```

Любые другие файлы в `shared/` нужно импортировать вручную через алиас `#shared` (Nuxt настраивает его автоматически):

```ts
// For files directly in the shared directory
import capitalize from '#shared/capitalize'

// For files in nested directories
import lower from '#shared/formatters/lower'

// For files nested in a folder within utils
import upper from '#shared/utils/formatters/upper'
```

Алиас даёт единообразные импорты по всему приложению независимо от расположения файла.
