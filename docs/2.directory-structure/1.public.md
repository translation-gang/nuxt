Файлы в каталоге `public/` отдаются с корня и не изменяются процессом сборки. Подходит для файлов, которым важны имена (например, `robots.txt`) _или_ которые редко меняются (например, `favicon.ico`).

```bash [Directory structure]
-| public/
---| favicon.ico
---| og-image.png
---| robots.txt
```

```vue [app/app.vue]
<script setup lang="ts">
useSeoMeta({
  ogImage: '/og-image.png',
})
</script>
```

::tip{to="https://v2.nuxt.com/docs/directory-structure/static/" target="_blank"}
В Nuxt 2 это называется каталог [`static/`].
::
