Файлы из `public/` попадают в корень отдачи сервера и не проходят через сборщик. Так удобно хранить ресурсы с фиксированными именами (`robots.txt`, `favicon.ico` и т.п.).

```bash [Структура директории]
-| public/
---| favicon.ico
---| og-image.png
---| robots.txt
```

```vue [app.vue]
<script setup lang="ts">
useSeoMeta({
  ogImage: '/og-image.png'
})
</script>
```

::tip{to="https://v2.nuxt.com/docs/directory-structure/static" target="_blank"}
В Nuxt 2 это была директория [`static/`].
::
