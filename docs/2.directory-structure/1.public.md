Файлы из директории `public/` отдаются по корневому пути и не обрабатываются сборкой. Подходит для файлов, которым нужно сохранить имя (например, `robots.txt`) _или_ которые редко меняются (например, `favicon.ico`).

```bash [Структура каталогов]
-| public/
---| favicon.ico
---| og-image.png
---| robots.txt
```

```vue [app/app.vue]
<script setup lang="ts">
useSeoMeta({
  ogImage: '/og-image.png',
})
</script>
```

::tip{to="https://v2.nuxt.com/docs/directory-structure/static/" target="_blank"}
В Nuxt 2 эта директория называлась [`static/`].
::
