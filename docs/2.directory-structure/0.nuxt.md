::important
Добавьте этот каталог в [`.gitignore`](/docs/3.x/directory-structure/gitignore), чтобы не коммитить артефакты dev-сборки.
::

Этот каталог полезен, чтобы увидеть, что Nuxt генерирует из структуры проекта.

Nuxt также даёт виртуальную файловую систему (VFS) для модулей: шаблоны можно подмешивать в `.nuxt/` без записи на диск.

Вы можете изучить сгенерированные файлы, открыв [Nuxt DevTools](https://devtools.nuxt.com) в режиме разработки и перейдя на вкладку **Виртуальные файлы**.

::warning
Не изменяйте файлы в этой директории: при запуске [`nuxt dev`](/docs/3.x/api/commands/dev) она пересоздаётся целиком.
::
