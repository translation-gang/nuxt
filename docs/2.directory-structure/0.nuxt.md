::important
Эту директорию стоит добавить в [`.gitignore`](/docs/4.x/directory-structure/gitignore), чтобы не коммитить артефакты сборки в репозиторий.
::

Директория полезна, если нужно разобраться, какие файлы Nuxt генерирует по вашей структуре директорий.

Nuxt также предоставляет виртуальную файловую систему (VFS) для модулей: они могут добавлять шаблоны в эту директорию без записи на диск.

Сгенерированные файлы можно посмотреть в [Nuxt DevTools](https://devtools.nuxt.com) в режиме разработки на вкладке **Virtual Files**.

::warning
Не изменяйте файлы внутри: вся директория пересоздаётся при запуске [`nuxt dev`](/docs/4.x/api/commands/dev).
::
