::important
Этот каталог нужно добавить в [`.gitignore`](/docs/4.x/directory-structure/gitignore), чтобы не отправлять в репозиторий вывод dev-сборки.
::

Этот каталог полезен, если вы хотите подробнее разобраться в файлах, которые Nuxt генерирует из структуры проекта.

Nuxt также предоставляет виртуальную файловую систему (VFS), чтобы модули могли добавлять шаблоны в этот каталог без записи на диск.

Сгенерированные файлы можно посмотреть, открыв [Nuxt DevTools](https://devtools.nuxt.com) в режиме разработки и перейдя на вкладку **Virtual Files**.

::warning
Не изменяйте файлы внутри: весь каталог пересоздаётся при запуске [`nuxt dev`](/docs/4.x/api/commands/dev).
::
