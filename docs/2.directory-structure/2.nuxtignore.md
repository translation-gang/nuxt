Файл `.nuxtignore` заставляет Nuxt игнорировать файлы в корне проекта ([`rootDir`](/docs/4.x/api/nuxt-config#rootdir)) при сборке.

Используется тот же синтаксис, что в [`.gitignore`](/docs/4.x/directory-structure/gitignore) и `.eslintignore`: каждая строка — glob-шаблон игнорируемых файлов.

::tip
В `nuxt.config` можно настроить [`ignoreOptions`](/docs/4.x/api/nuxt-config#ignoreoptions), [`ignorePrefix`](/docs/4.x/api/nuxt-config#ignoreprefix) и [`ignore`](/docs/4.x/api/nuxt-config#ignore).
::

## Использование

```bash [.nuxtignore]
# игнорировать лейаут foo.vue
app/layouts/foo.vue
# игнорировать лейауты с суффиксом -ignore.vue
app/layouts/*-ignore.vue

# игнорировать страницу bar.vue
app/pages/bar.vue
# игнорировать страницы в папке ignore
app/pages/ignore/*.vue

# игнорировать файлы мидлвара в foo, кроме foo/bar.js
app/middleware/foo/*.js
!app/middleware/foo/bar.js
```
