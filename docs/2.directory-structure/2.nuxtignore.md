Файл `.nuxtignore` говорит Nuxt не учитывать файлы в корне проекта ([`rootDir`](/docs/4.x/api/nuxt-config#rootdir)) при сборке.

Действует по тем же правилам, что [`.gitignore`](/docs/4.x/directory-structure/gitignore) и `.eslintignore`: каждая строка — glob-паттерн игнорируемых файлов.

::tip
Также можно настроить [`ignoreOptions`](/docs/4.x/api/nuxt-config#ignoreoptions), [`ignorePrefix`](/docs/4.x/api/nuxt-config#ignoreprefix) и [`ignore`](/docs/4.x/api/nuxt-config#ignore) в `nuxt.config`.
::

## Использование

```bash [.nuxtignore]
# ignore layout foo.vue
app/layouts/foo.vue
# ignore layout files whose name ends with -ignore.vue
app/layouts/*-ignore.vue

# ignore page bar.vue
app/pages/bar.vue
# ignore page inside ignore folder
app/pages/ignore/*.vue

# ignore route middleware files under foo folder except foo/bar.js
app/middleware/foo/*.js
!app/middleware/foo/bar.js
```
