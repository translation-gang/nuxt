::important
Этот каталог нужно добавить в [`.gitignore`](/docs/4.x/directory-structure/gitignore), чтобы не отправлять в репозиторий артефакты сборки.
::

Используйте этот каталог для деплоя приложения Nuxt в production.

:read-more{to="/docs/4.x/getting-started/deployment"}

::warning
Не изменяйте файлы внутри: весь каталог пересоздаётся при запуске [`nuxt build`](/docs/4.x/api/commands/build).
::
