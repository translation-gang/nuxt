::important
Эту директорию стоит добавить в [`.gitignore`](/docs/4.x/directory-structure/gitignore), чтобы не коммитить артефакты сборки в репозиторий.
::

Используйте эту директорию для развёртывания приложения Nuxt в продакшен.

:read-more{to="/docs/4.x/getting-started/deployment"}

::warning
Не изменяйте файлы внутри: вся директория пересоздаётся при запуске [`nuxt build`](/docs/4.x/api/commands/build).
::
