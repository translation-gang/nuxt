Файл `.gitignore` задаёт, какие файлы git должен игнорировать.

:read-more{icon="i-simple-icons-git" title="документация git" to="https://git-scm.com/docs/gitignore" target="_blank"}

Рекомендуем **как минимум** такие записи:

```bash [.gitignore]
# Nuxt dev/build outputs
.output
.data
.nuxt
.nitro
.cache
dist

# Node dependencies
node_modules

# Logs
logs
*.log

# Misc
.DS_Store

# Local env files
.env
.env.*
!.env.example
```
