[Nuxt Content](https://content.nuxt.com) читает каталог `content/` и разбирает файлы `.md`, `.yml`, `.csv` и `.json`, формируя файловую CMS.

- Рендер контента встроенными компонентами.
- Запросы к контенту через API в духе MongoDB.
- Vue-компоненты в Markdown через синтаксис MDC.
- Автоматическая генерация навигации.

::read-more{to="https://content.nuxt.com" target="_blank"}
Подробнее в документации **Nuxt Content**.
::

## Подключение Nuxt Content

Установите модуль `@nuxt/content` и добавьте его в `nuxt.config.ts` одной командой:

```bash [Terminal]
npx nuxt module add content
```

## Создание контента

Поместите markdown-файлы в `content/`:

```md [content/index.md]
# Hello Content
```

Модуль загрузит и разберёт их автоматически.

## Рендер контента

Для страниц контента добавьте [catch-all маршрут](/docs/4.x/directory-structure/app/pages/#catch-all-route) с компонентом [`<ContentRenderer>`](https://content.nuxt.com/docs/components/content-renderer):

```vue [app/pages/[...slug\\].vue]
<script lang="ts" setup>
const route = useRoute()
const { data: page } = await useAsyncData(route.path, () => {
  return queryCollection('content').path(route.path).first()
})
</script>

<template>
  <div>
    <header><!-- ... --></header>

    <ContentRenderer
      v-if="page"
      :value="page"
    />

    <footer><!-- ... --></footer>
  </div>
</template>
```
