---
title: "Автоимпорты"
description: "В Nuxt 3 компоненты и компосаблы по возможности автоимпортируются."
---

::note
В остальной документации по миграции у утилит Nuxt и Vue нет явных импортов — это не опечатка. Nuxt подставляет их автоматически; подсказки типов работают при включённой [поддержке TypeScript](/docs/4.x/migration/configuration#typescript).
::

[Подробнее об автоимпортах](/docs/4.x/guide/concepts/auto-imports)

## Миграция

1. Если в Nuxt 2 использовали `@nuxt/components`, уберите `components: true` из `nuxt.config`. При более сложной настройке учтите, что опции компонентов изменились. См. [документацию по компонентам](/docs/4.x/directory-structure/app/components).

::tip
Разрешение компонентов и автоимпортов композаблов можно посмотреть в `.nuxt/types/components.d.ts` и `.nuxt/types/imports.d.ts`.
::
