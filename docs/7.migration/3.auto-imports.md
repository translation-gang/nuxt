---
title: Автоимпорт
description: В Nuxt 3 по возможности компоненты и композаблы подключаются автоматически — меньше ручного импорта.
---

::note
В остальной документации по миграции у ключевых утилит Nuxt и Vue может не быть явных импортов. Это не опечатка: Nuxt импортирует их сам; подсказки типов появятся, если вы настроили TypeScript по [инструкции](/docs/4.x/migration/configuration#typescript).
::

[Подробнее об автоимпорте](/docs/4.x/guide/concepts/auto-imports)

## Миграция

1. Если в Nuxt 2 использовался `@nuxt/components`, удалите `components: true` из `nuxt.config`. При более сложной настройке учтите, что опции компонентов изменились. См. [документацию по компонентам](/docs/4.x/directory-structure/app/components).

::tip
Посмотрите `.nuxt/types/components.d.ts` и `.nuxt/types/imports.d.ts`, чтобы увидеть, как Nuxt разрешил автоимпорт компонентов и композаблов.
::
