В Nuxt 3 мета-теги можно задавать так:
1. Через `nuxt.config`.
2. Через композабл [`useHead`](/docs/4.x/api/composables/use-head) ([документация](/docs/4.x/getting-started/seo-meta))
3. Через [глобальные meta-компоненты](/docs/4.x/getting-started/seo-meta)

Доступны `title`, `titleTemplate`, `base`, `script`, `noscript`, `style`, `meta`, `link`, `htmlAttrs` и `bodyAttrs`.

::tip
Сейчас для мета-тегов используется [`Unhead`](https://github.com/unjs/unhead); детали реализации могут меняться.
::

:read-more{to="/docs/4.x/getting-started/seo-meta"}

## Миграция

1. В `nuxt.config` переименуйте `head` в `meta`. Общую конфигурацию мета лучше вынести в `app.vue`. (У объектов больше нет ключа `hid` для дедупликации.)
2. Если к состоянию компонента обращались через `head`, перейдите на [`useHead`](/docs/4.x/api/composables/use-head) или встроенные meta-компоненты.
3. В Options API можно использовать метод `head()` вместе с `defineNuxtComponent`.

### useHead

::code-group

```vue [Nuxt 2]
<script>
export default {
  data: () => ({
    title: 'My App',
    description: 'My App Description',
  }),
  head () {
    return {
      title: this.title,
      meta: [{
        hid: 'description',
        name: 'description',
        content: this.description,
      }],
    }
  },
}
</script>
```

```vue [Nuxt 3]
<script setup lang="ts">
const title = ref('My App')
const description = ref('My App Description')

// This will be reactive when you change title/description above
useHead({
  title,
  meta: [{
    name: 'description',
    content: description,
  }],
})
</script>
```

::

### Meta-компоненты

В Nuxt 3 есть meta-компоненты для той же задачи. Они похожи на HTML-теги, но предоставляются Nuxt и работают аналогично.

::code-group

```vue [Nuxt 2]
<script>
export default {
  head () {
    return {
      title: 'My App',
      meta: [{
        hid: 'description',
        name: 'description',
        content: 'My App Description',
      }],
    }
  },
}
</script>
```

```vue [Nuxt 3]
<template>
  <div>
    <Head>
      <Title>My App</Title>
      <Meta
        name="description"
        content="My app description"
      />
    </Head>
    <!-- -->
  </div>
</template>
```

::

::important
1. Имена этих компонентов пишите с заглавной буквы, чтобы отличать от нативных HTML (`<Title>`, а не `<title>`).
2. Их можно размещать в любом месте шаблона страницы.
::

### Options API

```vue [Nuxt 3 (Options API)]
<script>
// if using options API `head` method you must use `defineNuxtComponent`
export default defineNuxtComponent({
  head (nuxtApp) {
    // `head` receives the nuxt app but cannot access the component instance
    return {
      meta: [{
        name: 'description',
        content: 'This is my page description.',
      }],
    }
  },
})
</script>
```
