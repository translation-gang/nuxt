---

В собранном приложении Nuxt 3 нет runtime-зависимости Nuxt: сайт быстрее и легче, но хуки runtime-сервера Nuxt больше недоступны.

:read-more{to="/docs/4.x/guide/concepts/server-engine"}

## Шаги

1. Удалите ключ `render` из `nuxt.config`.
2. Файлы в `~~/server/api` и `~~/server/middleware` регистрируются автоматически; уберите их из массива `serverMiddleware`.
3. Остальные элементы `serverMiddleware` должны указывать на файлы или npm-пакеты напрямую, а не на inline-функции.
