---
title: Обзор
description: Nuxt 3 является полностью переписанной версией Nuxt 2, а также базируется на новом наборе базовых технологий.
---

При миграции приложения с Nuxt 2 на Nuxt 3 меняется многое; по мере созревания экосистемы сам процесс миграции обычно упрощается.

::note
Это руководство по миграции дополняется и согласуется с развитием Nuxt 3.
::

Некоторые из этих значительных изменений включают:

1. Переход от Vue 2 к Vue 3, включая использование по умолчанию Composition API и синтаксиса `<script setup>`.
2. Переход от webpack 4 и Babel к Vite или webpack 5 и esbuild.
3. Переход от зависимости Nuxt во время выполнения к минимальному автономному серверу, собранному с помощью Nitro.

::tip
Если вам необходимо остаться на Nuxt 2, но вы хотите воспользоваться преимуществами Nuxt 3 в Nuxt 2, вы можете также ознакомиться с [руководством по началу работы с Bridge](/docs/bridge/overview).
::
