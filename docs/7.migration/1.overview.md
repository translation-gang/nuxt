При переходе с Nuxt 2 на Nuxt 3 многое меняется, хотя миграция по мере стабилизации становится проще.

::note
Этот гайд по миграции дополняется по мере развития Nuxt 3.
::

Основные изменения:

1. Переход с Vue 2 на Vue 3, включая Composition API и script setup по умолчанию.
2. Переход с webpack 4 и Babel на Vite или webpack 5 и esbuild.
3. Переход с runtime-зависимости Nuxt на минимальный автономный сервер, собираемый Nitro.

::tip
Если нужно остаться на Nuxt 2, но использовать часть возможностей Nuxt 3, см. [начало работы с Bridge](/docs/4.x/bridge/overview).
::
