При миграции приложения с Nuxt 2 на Nuxt 3 изменений много, хотя по мере выхода стабильных релизов процесс должен становиться проще.

::note
Это руководство по миграции дорабатывается в соответствии с развитием Nuxt 3.
::

К числу важных изменений относятся:

1. Переход с Vue 2 на Vue 3, включая Composition API и `<script setup>` по умолчанию.
1. Переход с webpack 4 и Babel на Vite или webpack 5 и esbuild.
1. Переход от runtime-зависимости Nuxt к минимальному автономному серверу, собираемому с nitropack.

::tip
Если нужно остаться на Nuxt 2, но получить возможности Nuxt 3, см. [как начать работу с Bridge](/docs/4.x/bridge/overview).
::
