# E3009

`useAsyncData()` вызвали без функции-обработчика. Обработчик выполняет fetch и возвращает данные — его нельзя опустить или передать другим типом.

## Решение

```ts
const { data } = useAsyncData('users', () => $fetch('/api/users'))
```
