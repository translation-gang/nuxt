# E4012

Nuxt не смог разобрать ответ при рендере server component (island). Обычно endpoint server component вернул не ожидаемый island payload — например, страницу ошибки или некорректный HTML.

## Решение

Проверьте server component на ошибки, мешающие рендеру, и убедитесь, что island endpoint возвращает валидный ответ. Посмотрите сетевой ответ на запрос island — что именно вернул сервер.
