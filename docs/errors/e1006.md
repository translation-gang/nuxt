# E1006

`onPrehydrate()` выполнился без трансформации сборочным конвейером Nuxt. Ему нужна обработка на этапе компиляции, и он работает только на сервере — ошибка возникает, когда вызов идёт из зависимости, которую Nuxt не транспилирует.

## Решение

Добавьте проблемную библиотеку в `build.transpile` в `nuxt.config.ts`, чтобы сборочный конвейер обработал её вызов `onPrehydrate()`.

::
