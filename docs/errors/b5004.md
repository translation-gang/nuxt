# B5004

Nuxt обнаружил отдельный файл `vite.config` или `webpack.config` рядом с `nuxt.config`. Nuxt управляет сборщиком внутри себя и игнорирует такие файлы — обычно это остатки после миграции или скопированный не-Nuxt проект.

## Решение

Перенесите конфигурацию в `nuxt.config` под соответствующий ключ и удалите внешний файл:

- `vite.config`: используйте ключ `vite`
- `webpack.config`: используйте ключ `webpack`
- `nitro.config`: используйте ключ `nitro`
- `postcss.config`: используйте ключ `postcss`

```ts
export default defineNuxtConfig({
  vite: {
    // your Vite config here
  },
})
```
