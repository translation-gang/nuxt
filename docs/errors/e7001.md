# E7001

Запросили payload URL с полным адресом, включая hostname. Payload всегда отдаётся с того же origin, что и приложение — URL с hostname указывает наружу и отклоняется.

## Решение

Передайте относительный путь вместо полного URL:

```ts
loadPayload('/some-page')
```
