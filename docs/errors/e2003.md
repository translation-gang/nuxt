# E2003

`abortNavigation()` вызвали вне route middleware. Отменить навигацию можно только из обработчика middleware — вызов из компонента, плагина, composable или колбэка без контекста middleware завершится ошибкой.

## Решение

Перенесите вызов `abortNavigation()` в тело обработчика `defineNuxtRouteMiddleware()`.
