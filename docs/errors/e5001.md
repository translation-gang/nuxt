# E5001

Код, зависящий от app manifest, выполнился при выключенном `experimental.appManifest`. Manifest нужен для route rules и определения prerendered payload на клиенте — без него эти возможности не работают.

## Решение

Включите app manifest в `nuxt.config`:

```ts
export default defineNuxtConfig({
  experimental: {
    appManifest: true,
  },
})
```
