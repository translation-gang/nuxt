# E3001

URL, переданный в `useFetch()`, начинается с `//`. Протокол-относительный URL разрешается во внешний хост — почти никогда это не задумано, поэтому такие запросы отклоняются, чтобы не утечь на другой origin.

## Решение

Используйте абсолютный URL с явным протоколом или относительный путь:

```ts
useFetch('/api/data')
useFetch('https://api.example.com/data')
```
