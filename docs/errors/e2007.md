# E2007

`setPageLayout()` вызвали из `setup()` компонента во время SSR. На сервере layout нужно определить до рендера компонента — изменение здесь даёт некорректный SSR-вывод.

## Решение

Задайте layout в route middleware или статически через `definePageMeta({ layout: '...' })`.
