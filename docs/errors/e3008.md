# E3008

`useAsyncData()` вызвали без корректного ключа. Первый аргумент — непустая строка, чтобы Nuxt мог кэшировать запрос и дедуплицировать его между компонентами.

## Решение

```ts
const { data } = useAsyncData('users', () => $fetch('/api/users'))
```
