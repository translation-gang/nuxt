# B5003

У вас есть пользовательские ключи в `runtimeConfig.app` — пространстве имён, которое Nuxt резервирует для внутренних значений (например, `baseURL` и `cdnURL`). Пользовательские ключи здесь могут конфликтовать с конфигурацией самого Nuxt.

## Решение

Перенесите пользовательские ключи в `runtimeConfig.public` (доступны на клиенте) или в отдельное пространство имён верхнего уровня (только на сервере):

```ts
export default defineNuxtConfig({
  runtimeConfig: {
    // instead of runtimeConfig.app.myKey
    public: {
      myKey: 'value',
    },
  },
})
```
