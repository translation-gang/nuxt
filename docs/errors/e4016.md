# E4016

Текущий маршрут совпадает с вложенной страницей, но родительский page-компонент не рендерит `<NuxtPage />`. Vue Router показывает дочернюю страницу только в `<NuxtPage />` родителя — без него дочерняя страница не появится.

Так бывает при структуре каталогов ниже, если в `parent.vue` нет `<NuxtPage />`:

```bash [Directory Structure]
-| pages/
---| parent/
------| child.vue
---| parent.vue
```

## Решение

Добавьте `<NuxtPage />` в родительский page-компонент, чтобы дочерний маршрут рендерился внутри него. Если вложенный маршрут не нужен, перестройте каталог `pages/`, чтобы у страницы не было дочерних маршрутов.

::
