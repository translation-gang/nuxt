# E7008

`callOnce()` получил аргумент `fn`, который не является функцией. Код, который нужно выполнить один раз, передают функцией — тогда Nuxt выполнит его единожды и пропустит при следующих вызовах.

## Решение

```ts
await callOnce('setup', () => {
  // runs once
})
```
