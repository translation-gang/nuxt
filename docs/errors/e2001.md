# E2001

`navigateTo()` получил внешний URL без опции `{ external: true }`. Nuxt требует явного согласия на внешнюю навигацию, чтобы случайно не увести пользователя с вашего приложения.

## Решение

```ts
navigateTo('https://example.com', { external: true })
```
