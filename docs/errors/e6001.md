# E6001

Head composable, например `useHead()`, вызвали без активного экземпляра Unhead. Так бывает вне валидного контекста Nuxt — например, в оторванном асинхронном колбэке, когда экземпляр Nuxt уже недоступен.

## Решение

Вызывайте head composables синхронно в `setup()` компонента, плагине Nuxt или route middleware. Если нужен вызов после `await`, сначала захватите контекст Nuxt:

```ts
const nuxtApp = useNuxtApp()
await someAsyncWork()
nuxtApp.runWithContext(() => useHead({ title: 'Late title' }))
```
