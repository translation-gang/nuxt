# E2005

`useRoute()` вызвали внутри route middleware (напрямую или через другой composable). На этом этапе целевой маршрут ещё не разрешён, и `useRoute()` может вернуть неожиданные значения; middleware получает маршруты `to` и `from` аргументами.

## Решение

```ts
export default defineNuxtRouteMiddleware((to, from) => {
  // use `to` / `from` instead of useRoute()
})
```
