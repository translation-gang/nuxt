# E1001

Composable, которому нужен экземпляр Nuxt (`useNuxtApp()`, `useRoute()`, `useFetch()`, …), выполнился вне плагина, хука Nuxt, route middleware или Vue `setup()`. Чаще всего это вызов внутри асинхронного колбэка (`setTimeout`, `.then()`, после `await`), где контекст уже потерян.

## Решение

Вызывайте composable синхронно в начале `setup()` / плагина / middleware и переиспользуйте полученный результат позже. Для серверной асинхронной работы оберните вызов в `nuxtApp.runWithContext()`.
